## 3. 데이터 조인 규칙

### 조인 키
```
일 + 캠페인 + 그룹 + 소재
```

### 지표 우선순위

| 지표 | 기준 소스 | 비고 |
|------|-----------|------|
| 노출 | 채널(CH) 단독 | AF에 없음 |
| 비용 | 채널(CH) 단독 | AF에 없음 |
| 클릭 | CH / AF 병렬 비교 | 차이 = 어트리뷰션 윈도우 영향 |
| 회원가입 | CH / AF 병렬 비교 | |
| 구매 | CH / AF 병렬 비교 | |
| 구매매출 | CH / AF 병렬 비교 | |
| ROAS 계산 | 비용=CH, 전환=AF | 공식 지표는 AF 기준 |
| CPA 계산 | 비용=CH, 구매=AF | 공식 지표는 AF 기준 |

### 조인 후 컬럼 네이밍
- 채널 전환 지표: `클릭_ch`, `구매_ch`, `구매매출_ch`
- AF 전환 지표: `클릭_af`, `구매_af`, `구매매출_af`

## 4. 캠페인 구조

### 캠페인 목적 분류
```
플러스가입   → 신규 멤버십 가입 유도 (퍼널: 전환)
첫구매       → 첫 구매 전환 (퍼널: 전환)
리타겟팅     → 이탈 유저 재방문 (퍼널: 고려)
신규유저     → 신규 유저 획득 (퍼널: 인지)
룩얼라이크   → 유사 타겟 확장 (퍼널: 인지)
재구매       → 기존 유저 반복 구매 (퍼널: 재구매)
브랜드KW    → 브랜드 검색어 (퍼널: 고려)
일반KW      → 일반 검색어 (퍼널: 고려)
```

### 타겟 그룹 분류
```
논타겟    → 타겟 설정 없는 광범위 노출
유사타겟  → 기존 고객 유사 타겟 (Lookalike)
리마케팅  → 사이트/앱 방문 이력 기반
VIP       → 고가치 기존 고객
윈백      → 이탈 후 복귀 타겟
```

## 5. 소재 네이밍 컨벤션

### 형식
```
{소재타입}_{메시지}_{시즌}_{AB테스트}_{버전}
```

### 소재 타입
```
VID  → 동영상
IMG  → 이미지 (단일)
CRS  → 캐러셀 (복수 이미지)
TXT  → 텍스트 (검색광고)
```

### AB 테스트 구분
```
_A_ → A안 (대조군 또는 첫 번째 소재)
_B_ → B안 (실험군 또는 두 번째 소재)
없음 → 단일 소재 (비교 없음)
```

### 버전
```
v1 → 최초 버전
v2 → 수정 버전 (메시지·디자인 변경)
```

### 시즌 코드 (현재)
```
겨울 → 12월~2월 시즌 소재
```

### 파싱 예시
```
VID_플러스멤버십_겨울_A_v1
→ 소재타입: VID (동영상)
→ 메시지: 플러스멤버십
→ 시즌: 겨울
→ AB: A안
→ 버전: v1

CRS_특가_겨울_v1
→ 소재타입: CRS (캐러셀)
→ 메시지: 특가
→ 시즌: 겨울
→ AB: 없음 (단일)
→ 버전: v1
```

## 6. KPI 기준값 (벤치마크)

> ⚠️ 아래 값은 초안이며, 실제 기준값으로 업데이트 필요

| 지표 | 목표 | 경고선 | 기준 소스 |
|------|------|--------|-----------|
| ROAS | 300% 이상 | 100% 미만 | AF 기준 |
| CPA | ₩15,000 이하 | ₩30,000 초과 | AF 기준 |
| CTR | 1~5% | 0.5% 미만 | 채널 기준 |
| CVR (클릭→구매) | 3% 이상 | 1% 미만 | AF 기준 |

## 7. 데이터 품질 규칙

### 이상치 처리
```
클릭 > 노출         → 데이터 오류, 해당 행 제외
구매매출 / 구매 < 0 → 환불 건, 별도 플래그 처리
비용 = 0이고 클릭 > 0 → 오가닉 유입 가능성, 주석 표기
AF 클릭 / CH 클릭 > 2배 또는 < 0.3배 → 어트리뷰션 이슈, 주의 플래그
```

### 결측치 처리
```
AF 미매칭 행 (left join 결과 NA) → CH 지표만 사용, AF 지표는 0으로 표시하지 않고 NA 유지
비용 NA → 집계에서 제외
```

## 8. 분석 주의사항

- **VIP / 윈백 그룹은 모수가 작아** 단기 ROAS 변동이 크다. 최소 7일 이상 누적 데이터로 판단한다.
- **논타겟 vs 유사타겟 비교**는 동일 캠페인 목적 내에서만 유효하다.
- **채널(CH) 전환 vs AF 전환 차이**는 어트리뷰션 윈도우 설정 차이에서 발생한다. 두 수치를 단순 합산하지 않는다.
- **네이버(자체 채널)**는 외부 채널과 직접 ROAS 비교 시 주의. 브랜드 인지 효과가 포함된다.

## 10. 작업 원칙

- 분석 결과 출력 시 **CH 기준 / AF 기준** 명시 필수
- 추정값·집계값은 정확도 등급 표기: `정확 / 추정 / 오차 ±N%`
- 소재 타입·AB 구분은 네이밍 컨벤션 파싱으로 자동 분류, 수동 입력 금지
- 새 캠페인·소재 추가 시 이 문서의 컨벤션 섹션 먼저 업데이트
